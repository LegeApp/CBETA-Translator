# Bundled Fonts (Optional but Recommended)

For portable, print-quality bilingual PDFs, place open-source fonts here:

- `NotoSerifCJKtc-Regular.otf` (Chinese body text)
- `EBGaramond-Regular.ttf` (English body text)

Optional fallback names recognized by the loader:

- `SourceHanSerifTC-Regular.otf`
- `NotoSerif-Regular.ttf`

The PDF generator will prefer these bundled files first, then fall back to system fonts.
